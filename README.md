# discreet-dynasties

Family legacy and stewardship — building quiet, enduring wealth across generations. Content spans estate planning, generational philosophy, and the discipline of maintaining what matters without fanfare. Flat, self-documenting, no noise.

## Tech Stack

| Layer | Technology |
| --- | --- |
| Framework | Next.js |
| Database | Neon (Postgres) via Prisma |
| Auth | NextAuth |
| AI | Anthropic SDK |
| Payments | Stripe |
| PDF | React-PDF |
| Storage | Vercel Blob |
| Hosting | Vercel |

## Route Structure

- `/` — Public content listing
- `/book/[slug]` — Book chapters
- `/companion` — Companion guide
- `/forge` — AI Forge
- `/admin` — Admin dashboard
- `/account` — User account

## Site-Specific Functionality

- Book chapter publishing and navigation
- Companion guide content
- AI Forge for content generation
- Stripe-powered subscriptions for premium content
- PDF generation via React-PDF
